1. Configuração do Ambiente:
        Problema: Como você pode verificar se o .NET SDK está corretamente instalado em
        seu sistema? Em um arquivo de texto ou Markdown, liste os comandos que podem
        ser usados para verificar a(s) versão(ões) do .NET SDK instalada(s), como remover e
        atualizar.
        R: Para verificar a versão do .NET SDK instalado "dotnet --version ou "dotnet --info"
        Para remover no linux "sudo dotnet --uninstall-sdk <version>" e no "windows dotnet --uninstall-sdk <version>"
        Para atualizar os comandos são "sudo apt-get update" e "sudo apt-get install dotnet-sdk"
